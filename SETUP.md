# LoveDem — Setup

## Install

```bash
cd lovdem
npm install
```

## Run

```bash
npx expo start
```

Press `i` for iOS simulator, `a` for Android.

## Stack

- **Expo SDK 52** + Expo Router v4 (file-based navigation)
- **NativeWind v4** (Tailwind CSS for React Native)
- **react-native-svg** + **react-native-svg-transformer** (inline SVG assets as components)
- **@expo-google-fonts/plus-jakarta-sans** (matching the original typography)
- **@react-native-community/slider** (Love Level update sheet)
